### Project Overview
This project deploys a Node.js/Express web application to a cloud server
using two approaches: traditional host-based deployment and Docker containerization.

### Technology Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Web Server:** Nginx (Reverse Proxy)
- **Containerization:** Docker, Docker Compose
- **Cloud:** AWS EC2 (Ubuntu)
